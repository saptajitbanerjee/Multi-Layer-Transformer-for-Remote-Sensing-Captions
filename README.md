# Multi-Layer-Transformer-for-Remote-Sensing-Captions
Multi-Layer Transformer for remote sensing image captioning — EfficientNetB7 backbone, SiLU activations, custom label smoothing. +107.9% BLEU-4 and -76.3% loss over single-layer baseline, validated zero-shot on real-world 2026 Iran War satellite imagery. IIT Madras group project (individual contribution scope).
